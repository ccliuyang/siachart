# SiaChart
> 可编程，可定制，跨平台，通用的金融图表。

- 内置lua虚拟机，可开发图表内策略
- 跨平台：（linux / windows / mac / android / ios）
- 指标/公式系统（已集成Ta-lib）
- 外观显示完全可定制（可通过Lua引擎自定制文本解析器做任意定制）
